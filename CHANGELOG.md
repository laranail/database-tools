# Changelog

All notable changes to `laranail/database-tools` are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-26

Initial stable release — standalone Laravel database utilities: UUID/ULID/Nanoid
model traits, id-type-aware schema macros, soft-archive + soft-delete history,
money/datetime casts, slugging, audit observers, backup/restore with a database
CLI, and inspection/pagination helpers. No dependency on other Laranail packages.
